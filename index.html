PROJECT LOG: BARCELÓ FC

¿Quiénes somos?

Somos el equipo Barceló FC del ciclo SMR1 de ALMI Bilbao. Este proyecto representa la culminación de 3 semanas de trabajo en el reto RobotALMI, integrando robótica autónoma, programación avanzada y despliegue de sistemas.

Nuestra Flota de Robots

01. Siguelíneas & Obstáculos

Arduino UNO

Diseñado para una respuesta rápida en pista. Utiliza una matriz de sensores infrarrojos para detectar el contraste entre blanco y negro, junto con una lógica de tiempos para realizar maniobras de evasión de obstáculos programadas.

firmware_follower.ino
if (lecturaIzq == LOW && lecturaDer == LOW) {
    motorIzquierdo.write(50); 
    motorDerecho.write(130);
} else if (lecturaIzq == HIGH) {
    motorIzquierdo.write(90); // Ajuste fino
}

02. Otto Ninja (Bípede)

Bluetooth / Nano

Un robot humanoide centrado en la movilidad. Implementa una comunicación Serial vía Bluetooth que permite el control inalámbrico desde un smartphone. Se calibraron los ángulos de los servos para garantizar el equilibrio dinámico.

ninja_bt_control.ino
void loop() {
  if (BT.available()) {
    char cmd = BT.read();
    if(cmd == 'w') caminar();
    if(cmd == 's') retroceder();
    if(cmd == ' ') detener();
  }
}

03. Araña (Hexápodo)

Servo Dynamics

Plataforma de locomoción compleja. Utiliza múltiples grados de libertad para simular el movimiento orgánico de un arácnido. Actualmente en fase de optimización de la cinemática de las patas.
